¿Como realizamos nuestro trabajo?

Pasos a seguir

  1. 1
  2. 2
  3. 3
  4. 4

  1. Cotización

    Realizamos un trabajo de evaluación, donde tomamos en cuenta el tiempo que conlleva el desarrollo del proyecto, diseño, personalización, requisitos y secciones del mismo para poder brindar el mejor precio para un producto de excelente calidad.

  2. Reunión Inicial

    Una vez aprobada la cotización, se avanza a la segunda etapa, donde el cliente da un anticipo del 50% y nos brinda toda la información necesaria para poder comenzar.

  3. Propuesta de Diseño

    En esta instancia se le brinda al cliente diversas propuestas de diseño, para que las pueda evaluar, una vez seleccionada la propuesta, también podrá dictar que cambios le gustaría realizar en esta.

  4. Aprobación

    Una vez aprobado el diseño, se prosigue a realizar los últimos detalles para que el producto quede funcional, al cliente se le envía toda la información de su proyecto por medio de correo electrónico.

  5. Entrega

    La entrega final del producto sea realiza junto al pago del restante 50%, una vez completado todo el proceso, usted contara con un sitio web de confianza, funcional y muy atractivo para sus futuros clientes!

¿Tenés dudas?

Preguntas Frecuentes

  • ¿Qué toman en cuenta para la cotización?

    Para la cotización del proyecto se toman en cuenta varios factores, como, por ejemplo, el tiempo que tomara el mismo, las tecnologías a usar y los requisitos que se pidan para la página.

  • ¿Qué recibo una vez completado el proceso?

    Una vez completado el proceso, usted recibe el proyecto completo, ya en línea y funcional, también cuenta con un sistema opcional de Soporte y Mantenimiento del mismo.

  • ¿Cómo se realiza el pago del proyecto?

    Una vez realizada la cotización y propuesta de diseño, se efectúa el primer pago del 50% de lo acordado y una vez entregado el proyecto, se termina de completar el pago con el otro 50%.

  • ¿Qué tecnologías usan?

    Siempre vamos a utilizar la que mejor se amolde al proyecto planteado.

  • ¿Las propuestas son personalizadas?

    Siempre las propuestas son personalizadas, las páginas son 100% únicas y realizadas completamente a gusto del cliente.

  • Una vez entregado el Proyecto ¿hay que abonar algo más?

    Tener una página web siempre cuenta con un abono fijo, ya sea mensual o anual, eso a gusto del propietario, estos gastos son el Host de la página y el dominio.

    ¿Qué es el host?
    El host es el sitio donde se almacena la página.

    ¿Qué es el dominio?
    El dominio es el nombre único que identifica a tu página en internet.

  • Si quiero realizar cambios en mi Página ¿A quién recurro?

    Además del servicio de desarrollo, contamos con el servicio de soporte y mantenimiento, en el cual nos encargamos completamente de los cambios y modificaciones que se quieran realizar en la página, como de igual forma si llega a surgir algún inconveniente en la misma.
